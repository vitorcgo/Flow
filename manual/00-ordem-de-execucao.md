# Ordem de Execucao

## Fase 1: Preparar repositorio

1. Criar repo `flowdesk`.
2. Criar monorepo:
   - `backend/`
   - `frontend/`
   - `docs/`
3. Adicionar `.gitignore`.
4. Confirmar que nenhuma pasta de ferramenta entra no repo.

## Fase 2: Backend

1. Criar Spring Boot.
2. Configurar PostgreSQL.
3. Criar migrations.
4. Criar auth.
5. Criar tickets.
6. Criar auditoria.
7. Criar Swagger.

## Fase 3: Frontend

1. Criar React TypeScript.
2. Criar design system.
3. Criar AppShell.
4. Criar telas.
5. Conectar API.

## Fase 4: Deploy

1. Subir banco.
2. Subir backend.
3. Subir frontend.
4. Configurar CORS.
5. Testar login demo.

## Fase 5: Portfolio

1. Tirar prints.
2. Gravar demo curta.
3. Atualizar README.
4. Adicionar case no portfolio.
