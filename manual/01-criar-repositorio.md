# Manual: Criar repositorio

## Passos

1. Criar repositorio no GitHub:

```text
flowdesk
```

2. Descricao:

```text
Sistema corporativo de chamados, Kanban, SLA e auditoria com Java Spring Boot, React TypeScript e PostgreSQL.
```

3. Criar estrutura:

```text
backend/
frontend/
docs/
```

4. Criar `.gitignore` com:

```gitignore
.env
.env.*
!.env.example
node_modules/
dist/
build/
target/
.idea/
.vscode/
.DS_Store
```

## Checklist

- Repo publico criado.
- README inicial criado.
- Nenhuma pasta local de ferramenta commitada.
