# Checklist antes de publicar

## Codigo

- [ ] Build backend passou.
- [ ] Build frontend passou.
- [ ] Testes principais passaram.
- [ ] Variaveis reais nao estao commitadas.
- [ ] `.env` esta ignorado.
- [ ] README explica como rodar.

## Banco

- [ ] Migrations versionadas.
- [ ] Seed demo separado.
- [ ] Indices criados para filtros.
- [ ] Dados sensiveis nao aparecem no seed.

## Seguranca

- [ ] Senhas hasheadas.
- [ ] JWT secret fora do repositorio.
- [ ] CORS restrito ao frontend.
- [ ] Rotas protegidas por role.
- [ ] Respostas nao vazam stack trace.

## Interface

- [ ] Mobile nao quebra.
- [ ] Tabelas tem overflow controlado.
- [ ] Formularios mostram erro legivel.
- [ ] Loading e empty states existem.
- [ ] Nenhuma tela parece template generico.

## Demo

- [ ] Link frontend abre.
- [ ] Link backend health responde.
- [ ] Login demo funciona.
- [ ] Prints atualizados.
- [ ] Copy do portfolio revisado.
