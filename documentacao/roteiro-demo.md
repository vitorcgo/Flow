# Roteiro de Demo

## Abertura

FlowDesk e uma ferramenta interna para controlar chamados com SLA, Kanban e auditoria.

## Fluxo

1. Login como manager.
2. Dashboard mostra tickets abertos e atrasados.
3. Criar ticket critico.
4. Ticket aparece na lista.
5. Mover para "Em andamento".
6. Adicionar comentario interno.
7. Abrir auditoria e mostrar evento.
8. Mostrar Swagger/OpenAPI.

## Fechamento

O projeto demonstra stack corporativa: Java, Spring Boot, React, TypeScript, PostgreSQL, JWT, RBAC, logs e deploy.
