# Copy para portfolio

## Titulo

FlowDesk

## Subtitulo curto

Sistema corporativo de chamados, SLA, auditoria e relatorios.

## Descricao curta

Plataforma full-stack para operacoes internas com controle de acesso, fluxo de atendimento, Kanban, indicadores e rastreabilidade de acoes.

## Descricao para card

Sistema B2B construindo uma experiencia parecida com ferramenta usada em empresa: login com permissoes, chamados por prioridade, SLA, auditoria, comentarios, relatorios e deploy separado entre frontend, backend e banco.

## Pontos fortes

- Java Spring Boot com API REST documentada.
- React TypeScript com interface operacional.
- PostgreSQL com modelagem relacional.
- RBAC, JWT, refresh token e sessao expirada.
- Logs estruturados, auditoria e status codes padronizados.
- Dashboard, Kanban, filtros, relatorios e seed demo.

## Stack para exibir

Java, Spring Boot, React, TypeScript, PostgreSQL, Docker, JWT, OpenAPI, Koyeb, Vercel, Aiven.

## Frase de contexto

Projeto criado para demonstrar dominio de sistemas internos corporativos, nao apenas telas isoladas.

## Observacao honesta

Projeto autoral de portfolio, com fluxo e arquitetura pensados para contexto empresarial.
