# 07 - Frontend base e design system

## Objetivo

Criar frontend React TypeScript com base visual corporativa, responsiva e sem aparencia generica.

## Antes de comecar

Ler:

- `flowdesk/frontend/README.md`
- `flowdesk/frontend/ux/design-system.md`
- `flowdesk/manual/03-frontend-react.md`
- `flowdesk/specs/product-spec.md`

## Tarefas

1. Criar app Vite React TypeScript em `frontend`.
2. Configurar rotas.
3. Criar API client.
4. Criar auth store.
5. Criar AppShell com sidebar, header e area principal.
6. Criar tokens visuais:

- cores
- tipografia
- espacamento
- bordas
- foco
- estados

7. Criar componentes base:

- Button
- Input
- Select
- Badge
- Table
- Tabs
- Modal utilitario
- Toast
- EmptyState
- LoadingState

8. Usar nomes de componentes, ids, classes CSS, funcoes e estados de dominio em portugues.
9. Comentarios de codigo, quando necessarios, devem ser em portugues.

## Direcao visual

- fundo branco ou neutro claro
- contraste bom
- bordas suaves
- tabelas caprichadas
- sidebar moderna
- densidade de ferramenta real
- nada de excesso decorativo

## Verificacao

Rodar:

```bash
npm install
npm run lint
npm run build
npm run dev
```

Checar desktop e mobile.

## Saida esperada

Frontend base pronto para receber telas.

Nao criar commit. Sugerir nomes de commit em portugues.
