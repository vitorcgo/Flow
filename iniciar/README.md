# Iniciar

Esta pasta organiza roteiros sequenciais para executar o FlowDesk sem perder contexto.

Use em ordem. Termine um roteiro, valide, depois siga para o proximo.

## Ordem

2. `01-iniciar-repositorio.md`
3. `02-backend-base.md`

13. `12-revisao-final.md`

## Como usar

Abra o arquivo da etapa atual, envie o conteudo para o executor e espere terminar.

Se aparecer bloqueio manual, resolva a acao indicada antes de seguir.

## Regra de ouro

Cada etapa deve terminar com:

- resumo do que foi feito
- arquivos alterados
- comandos de verificacao
- pendencias manuais, se houver
- proximo roteiro recomendado
